# C Maze Solver & Pathfinding Simulation 🧩

A comprehensive maze simulation developed in **C** that visualizes pathfinding algorithms. This project demonstrates the implementation of search algorithms to find the shortest path in a dynamic grid environment.

### 🧠 Key Features
- **Manual Drawing Mode:** Create your own custom walls and paths.
- **Random Maze Generation:** Procedural generation of maze structures.
- **Pathfinding Algorithms:** Visual implementation of **BFS (Breadth-First Search)** and **DFS (Depth-First Search)**.
- **Real-time Animation:** Watch how algorithms explore the maze and find the exit.

### 🛠️ Technical Details
- **Language:** C
- **Concepts:** Queues (for BFS), Stacks (for DFS), Recursion, and Win32 Console API for visualization.
- **IDE:** Dev-C++

### 🎮 How to Play
- Use **Arrow Keys** to move.
- **F1-F4:** Toggle between drawing, erasing, and wall modes.
- **F8:** Start the Pathfinding AI (BFS/DFS).
- **Space:** Pause/Resume the simulation.
